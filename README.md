# Driveflow Enterprises - JamboPay Payment Portal

Production-ready payment portal integrated with JamboPay API.

## Features
- ✅ Production JamboPay integration
- ✅ Secure payment processing
- ✅ Mobile-responsive design
- ✅ Real-time transaction tracking
- ✅ Deployed on GitHub Codespaces

## Setup
1. Open in GitHub Codespaces
2. The server starts automatically on port 8000
3. Access via the generated public URL

## Environment
- **Merchant**: Driveflow Enterprises Live Cred
- **API**: JamboPay Production
- **Port**: 8000

## Security
- Uses official JamboPay production API
- Secure credential handling
- Input validation and sanitization
